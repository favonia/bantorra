Module BantorraBasis__OS

type scheme =
| MacOS
| Linux
| Windows

The scheme refers to how the default directory should be determined when XDG_CONFIG_HOME or XDG_CACHE_HOME is missing.

It does not correspond to the actual OS that is running. For example, the Linux scheme covers all BSD-like systems and Cygwin on Windows.
