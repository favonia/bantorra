val replace_cache : t -> unitpath -> source_digest:Stdlib.Digest.t -> BantorraBasis.Marshal.t -> Stdlib.Digest.t

replace_cache m unitpath ~source_digest value replaces the cache associated with unitpath and source_digest with value. It returns the digest of the stored cache.

val find_cache_opt : t -> unitpath -> source_digest:Stdlib.Digest.t -> cache_digest:Stdlib.Digest.t option -> BantorraBasis.Marshal.t option

find_cache m unitpath ~source_digest ~cache_digest value finds the cache associated with unitpath and source_digest. If cache_digest is None, it means the digest checking is skipped. One should use the digest returned by replace_cache whenever possible.
