Hook for Library Managers

The following API is for a library manager to chain all the libraries together. Please use the high-level API in Manager instead.

val to_unitpath : global:(current_root:BantorraBasis.File.filepath -> Bantorra.Anchor.lib_ref -> unitpath -> t * unitpath) -> t -> unitpath -> t * unitpath

to_unitpath ~global lib unitpath resolves unitpath and returns the eventual library where the unit belongs and the local unit path pointing to the unit.

parameter global

The global resolver for unit paths pointing to other libraries.

val to_filepath : global:(current_root:BantorraBasis.File.filepath -> Bantorra.Anchor.lib_ref -> unitpath -> suffix:string -> t * BantorraBasis.File.filepath) -> t -> unitpath -> suffix:string -> t * BantorraBasis.File.filepath

to_filepath ~global lib unitpath ~suffix resolves unitpath and returns the eventual library where the unit belongs and the underlying file path of the unit. It is similar to to_unitpath but returns a file path instead of a unit path.

parameter global

The global resolver for unit paths pointing to other libraries.

parameter suffix

The suffix shared by all the units in the file system.
